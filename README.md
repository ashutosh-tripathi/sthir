# STHIR

STHIR is a minimalist, manual-first activity tracking app focused on intentional daily practice.

This repository contains the web implementation of STHIR and serves as the foundation for future iOS and Apple Watch apps.

---

## Tech Stack

- Frontend: React / Next.js
- Styling: Tailwind CSS
- State: Local-first (browser storage initially)
- Platform: Web (mobile-first)

---

## Getting Started

### Prerequisites
- Node.js (18+)
- npm or yarn

### Setup

```bash
git clone https://github.com/ashutosh-tripathi/sthir.git
cd sthir
npm install
npm run dev
